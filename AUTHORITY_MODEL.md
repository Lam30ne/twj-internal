# Authority + Review Cadence Model

This layer clarifies how each document should be used and when it should be revisited.

## Authority Levels

### Source of Truth
- Canonical guidance
- Should drive current decisions
- Must be kept current

### Supporting
- Provides frameworks, thinking, or supporting context
- Should not override active operating decisions

### Historical Record
- Captures a point in time (review, audit, meeting)
- Not intended for ongoing decision-making

### Personal Guidance
- Intended for a specific individual or restricted audience
- Highly sensitive

## Review Cadence

Each document includes:
- Last reviewed date
- Next review date

### Expectations
- Source of truth: reviewed quarterly
- Supporting: reviewed 1–2x per year
- Historical: no scheduled review
- Personal: reviewed as needed

## Key Rule

If a document is:
- NOT source of truth
- OR past its review date

…it should be treated with caution and cross-referenced with leadership.
